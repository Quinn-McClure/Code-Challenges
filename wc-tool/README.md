## How to use ##
The wc Tool is a command-line tool that can be used to get the word, line, character, or byte count of a file. To run, clone the repo and then compile the java code using: `javac *.java`

Then to run the project, type: `java wcTool [-flag] [filename]`

The flags implemented here are:
- -c: counts the number of bytes in the file
- -l: counts the number of lines in the file
- -w: counts the number of words in the file
- -m: counts the number of characters in the file
